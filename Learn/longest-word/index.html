    Given a sentence, your task is to return the longest word in the sentence.

    • Track the maximum length word

    3Ô∏è‚É£ Final solution

    
function LongestWord(sen) {
  const words = sen.replace(/[^a-zA-Z0-9 ]/g, '').split(' ');

  let maxi = 0;
  let res = "";

  words.forEach(item => {
    if (item.length > maxi) {
      res = item;
      maxi = item.length;
    }
  });

  return res;
}

    The solution runs in O(n) time, where n is the length of the sentence.
