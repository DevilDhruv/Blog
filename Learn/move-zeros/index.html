void moveZeroes(vector<int>& nums) {
    int n = nums.size();
    vector<int> temp(n, 0);
    int j = 0;

    for(int i = 0; i < n; i++) {
        if(nums[i] != 0) {

        }

    Time: O(n)

    Space: O(n)

    This works, but violates the in-place requirement. In interviews, this solution is acceptable only as a starting point.

    ‚ö° Approach 2: Two Pointer (Optimal)

    We use a pointer j to track where the next non-zero element should go. 

    
void moveZeroes(vector<int>& nums) {
    int n = nums.size();
    int j = 0;

    for(int i = 0; i < n; i++) {
        if(nums[i] != 0) {
            nums[j++] = nums[i];
        }
    }

    for(int i = j; i < n; i++) {
        nums[i] = 0;
    }
}

    Time: O(n)

    Space: O(1)

    This is the solution interviewers are looking for. It‚Äôs clean, efficient, and preserves order.
